# bpSp02-estimation-ippg

MATLAB iPPG pipeline + recorder app + Python BP inference server.

Folders:
- app/matlab: pipeline + recorder + utilities
- app/python_server: FastAPI inference server
- third_party: external upstream code (kept intact)
- models: model files (.h5) (optional; can be external)
- config: runtime configs
- data, outputs: local runtime artifacts (ignored by git)
